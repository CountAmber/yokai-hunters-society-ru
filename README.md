# Yokai Hunter Society — Русский перевод [ru]
Модуль локализации системы [Yokai Hunter Society](https://github.com/bernardoclon/yhs) для Foundry VTT на русский язык. Перевод адаптирует листы персонажей, характеристики и интерфейс системы.

## Об игре
**«Общество охотников на ёкаев»** — это ролевая игра с лёгкими правилами, основанная на системе *Tunnel Goons*. Действие происходит в Японии эпохи Мэйдзи (конец XIX века). Игроки берут на себя роли членов тайного общества, которые выслеживают и сражаются с японскими монстрами — ёкаями. Игра делает упор на атмосферу мистического детектива, психологического хоррора и быстрые, смертоносные схватки.

## Установка
1. Откройте Foundry VTT и перейдите во вкладку **Add-on Modules**.
2. Нажмите кнопку **Install Module**.
3. Вставьте следующую ссылку в поле **Manifest URL**:
   ```text
   [https://raw.githubusercontent.com/CountAmber/yokai-hunters-society-ru/main/module.json](https://raw.githubusercontent.com/CountAmber/yokai-hunters-society-ru/main/module.json)
4. Нажмите Install.
5. Зайдите в ваш мир, откройте Manage Modules и включите Yokai Hunter Society — Русский перевод.
6. Убедитесь, что в Configure Settings -> Core Settings -> Language Preference выбран Russian.
